# Домашнє завдання: події та критичні секції

Курс: **Системне програмування**  
Тема: **Події. Критичні секції**  
Модуль 4. Частина 2

## Опис

Проєкт реалізовано як консольний C# застосунок. Усі чотири завдання знаходяться в одному файлі `Program.cs`, а запуск виконується через меню.

## Реалізовані завдання

1. **Пари чисел, суми та добутки**  
   Перший потік генерує пари чисел і записує їх у файл. Другий і третій потоки очікують подію завершення генерації через `ManualResetEventSlim`, після чого окремо рахують суми та добутки й записують результати у файли.

2. **Кінцева автобусна зупинка з одним номером маршруту**  
   Потоки імітують прихід пасажирів та прибуття автобусів маршруту №175. Кількість людей на зупинці змінюється у критичній секції через `lock`.

3. **Автобуси з різними номерами**  
   Кожен пасажир приходить на конкретний номер маршруту. Якщо прибуває автобус іншого номера, пасажир залишається чекати потрібний маршрут.

4. **Не кінцева зупинка**  
   Автобус може приїхати вже з пасажирами у салоні. Посадка виконується тільки на кількість вільних місць.
